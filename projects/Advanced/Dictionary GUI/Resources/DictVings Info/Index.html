Basic Documentation of the App DictVings

Features

  1. A simple look up feature that fetches all the definitions available
  2. It can look up the words from a text file. It can fetch the definitions for each word and can save them to a file locally
  3. it retails the histroy of lookup words

Overview

  • It fetches the definitions from an API
  • Most of the exceptions and errors are properly handled
  • To be able to read from a text file, There should be one word in each line.
  • The words misspelt or with no available information are seperately saved to a list
  • When there are more than two definitions available for words, It saves all the definitions in a new window and offers to view all the definitions
  • The time loaded is based on asychronous programming at the moment. It will make use of multiple threads pretty soon
